# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in WaSP, please report it responsibly.

**Do NOT open a public issue.**

Instead, email: kobie3717@gmail.com

We will acknowledge your report within 48 hours and provide a detailed response within 5 business days.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.3.x   | Yes       |
| 0.2.x   | Yes       |
| < 0.2   | No        |
